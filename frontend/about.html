About Job Portal

A full-stack project built with FastAPI, PostgreSQL, and HTML/CSS

What is Job Portal?

Job Portal is a full-stack web application that connects job seekers with employers on a single platform. Job seekers can create profiles, search for opportunities, apply with cover letters, and track their application status in real time. Employers can post job listings, manage them, and review every application they receive.

This project was built as a final semester submission for the Open Source Software Development course at the University of Management and Technology (UMT), Lahore.

Technology Stack

Frontend HTML5, CSS3, JavaScript
Backend Python, FastAPI, SQLAlchemy
Database PostgreSQL on Supabase
Auth JWT Tokens, Bcrypt
Backend Deploy Render
Frontend Deploy Netlify
Key Features
🔐

Secure Authentication

JWT-based login and registration for both Job Seekers and Employers. Passwords are always hashed with bcrypt.

🔍

Smart Job Search

Search by keyword, filter by category, location, and job type. Results load in real time from the database.

📋

Application Tracking

Applicants can see their status: Pending, Reviewed, Shortlisted, or Rejected — updated by the employer.

🏢

Employer Tools

Employers can post, edit, and delete jobs. They can view every applicant and manage their status from one dashboard.

🔖

Job Bookmarking

Job seekers can save interesting listings to a personal bookmark list and come back to apply later.

📱

Responsive Design

The entire frontend adapts to mobile and desktop screens using CSS media queries. No framework required.
